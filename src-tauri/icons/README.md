# Tauri Icons

Place your app icons here:

- `32x32.png` - 32x32 pixels
- `128x128.png` - 128x128 pixels  
- `128x128@2x.png` - 256x256 pixels (for Retina displays)
- `icon.icns` - macOS icon bundle
- `icon.ico` - Windows icon

You can generate these from a single high-resolution image using:
- [tauri-apps/tauri-icons](https://github.com/tauri-apps/tauri-icons)
- `npm run tauri icon path/to/source-icon.png`

## Getting Started

To get started with development:

```sh
# Clone the repository
git clone <YOUR_GIT_URL>
cd hive-cad

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Technologies

- **Vite** + **React**
- **Tauri** (Rust Backend)
- **Three.js** / **React Three Fiber**
- **Replicad** (OCCT via WASM)
- **shadcn-ui** + **Tailwind CSS**
